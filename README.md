## Deskripsi Project
Repository ini dibuat untuk memenuhi tugas mata kuliah **Artificial Intelligence**.  
Project terdiri dari dua modul utama, yaitu:

- **Modul 1: Linear Regression**
- **Modul 2: Linear Classifier**

Seluruh implementasi dilakukan menggunakan **Python dari scratch** dengan metode **Stochastic Gradient Descent (SGD)** tanpa menggunakan library machine learning seperti scikit-learn.

---

## Modul 1: Linear Regression
### Case Study
Prediksi biaya medis personal berdasarkan **BMI (Body Mass Index)**.

### Dataset
**Medical Cost Personal Datasets**  
https://www.kaggle.com/datasets/mirichoi0218/insurance

### Variabel
- **Input (x):** bmi
- **Output (y):** charges

### Implementasi
- Membaca dataset menggunakan **pandas**
- Implementasi **SGD from scratch**
- Menghitung **MSE Loss**
- Visualisasi hasil regresi menggunakan **matplotlib**

---

## Modul 2: Linear Classifier
### Case Study
Klasifikasi harga ponsel berdasarkan spesifikasi.

### Dataset
**Mobile Price Classification**  
https://www.kaggle.com/datasets/iabhishekofficial/mobile-price-classification

### Fitur
- `ram`
- `battery_power`

### Target
- `price_range` (0–3)

### Implementasi
- Fungsi **softmax** menggunakan numpy
- **Cross-Entropy Loss**
- Training model menggunakan **SGD**
- Visualisasi **decision boundary**
